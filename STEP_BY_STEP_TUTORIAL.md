## 튜토리얼 1: 첫 로그인부터 기본 확인까지

### 📋 상황 설정
**오전 9시 30분, 출근 후 첫 업무로 시스템 상태를 확인해야 합니다.**

### 🎯 학습 목표
- Dashboard와 Google Sheet 접속 방법 익히기
- 전체 현황 파악하는 방법 배우기
- 우선순위 이슈 식별하는 방법 익히기

### 📱 단계별 실습

#### Step 1: Dashboard 접속하기
```
⏰ 소요시간: 1분

1. 브라우저 열기 (Chrome 권장)
2. 북마크에서 "P5 Dashboard" 클릭
   또는 URL 직접 입력: https://your-dashboard-url.com
3. 페이지 로딩 대기 (약 3초)
4. 메인 그리드가 표시되면 성공!
```

**✅ 확인 포인트**
- 12행 × 69열 그리드가 보이나요?
- 상단에 검색창이 있나요?
- 우측에 이슈 패널이 있나요?

#### Step 2: 전체 현황 한눈에 파악하기
```
⏰ 소요시간: 2분

1. 그리드 전체를 스캔하며 색상 확인
   🔴 빨간색: 긴급 이슈 (즉시 대응 필요)
   🟠 주황색: 중요 이슈 (당일 대응 필요)
   🟡 노란색: 일반 이슈 (계획적 대응)
   🟢 초록색: 정상 상태

2. 하단 통계 확인
   "전체 기둥: 828개, 선택됨: 0개"
   "이슈: Critical 2개, High 5개, Medium 12개"

3. 우측 이슈 패널에서 활성 이슈 개수 확인
```

**✅ 확인 포인트**
- 빨간색 기둥이 몇 개나 있나요?
- 전체 이슈 개수는 몇 개인가요?
- 가장 긴급한 이슈는 무엇인가요?

#### Step 3: Google Sheet에서 상세 정보 확인
```
⏰ 소요시간: 3분

1. 새 탭에서 Google Sheets 열기
2. "P5_복합동_메일분석_DB" 시트 열기
3. 가장 아래쪽(최신 데이터)으로 스크롤
4. 긴급도 컬럼(C열) 확인
5. "Critical" 또는 "Showstopper" 행 찾기
```

**✅ 확인 포인트**
- 최신 데이터가 언제 등록되었나요?
- Critical 이슈의 발생원은 어디인가요?
- AI 분석 내용이 이해되나요?

#### Step 4: 첫 번째 우선순위 이슈 선정
```
⏰ 소요시간: 2분

1. Dashboard로 돌아가기
2. 우측 이슈 패널에서 가장 위쪽 이슈 클릭
3. 이슈 상세 정보 확인:
   - 제목: 무엇에 대한 문제인가?
   - 영향 범위: 몇 개 기둥이 영향받나?
   - 발생원: 어느 업체에서 발생했나?
   - 긴급도: 얼마나 급한가?

4. 해당 기둥들이 그리드에서 하이라이트되는지 확인
```

**✅ 실습 완료 체크리스트**
- [ ] Dashboard 접속 성공
- [ ] 전체 현황 파악 완료
- [ ] Google Sheet 데이터 확인 완료
- [ ] 우선순위 이슈 1개 선정 완료

## 튜토리얼 2: 긴급 이슈 발견 시 대응 과정

### 📋 상황 설정
**Dashboard에서 빨간색 Critical 이슈를 발견했습니다. "PSRC 기둥 Shop Drawing 변경 요청 - A-X23~X30 구간"**

### 🎯 학습 목표
- 긴급 이슈 상세 정보 파악하기
- 관련자 연락 및 대응 과정 익히기
- 처리 상태 업데이트하는 방법 배우기

### 📱 단계별 실습

#### Step 1: 이슈 상세 정보 파악 (5분 이내)
```
⏰ 소요시간: 3분

1. Dashboard에서 해당 이슈 클릭
2. 상세 정보 확인:
   📍 위치: A-X23~X30 (8개 기둥)
   🏢 발생원: 삼우종합건축
   ⚠️ 긴급도: Critical
   📅 발생일: 2025-01-15 08:30
   📝 내용: Shop Drawing 변경으로 인한 철근 배치 수정 필요

3. Google Sheet에서 원본 메일 내용 확인:
   - 메일ID 복사
   - Sheet에서 Ctrl+F로 검색
   - "본문요약" 및 "AI분석" 컬럼 읽기
```

**✅ 파악해야 할 정보**
- [ ] 정확한 문제 내용
- [ ] 영향받는 기둥 개수와 위치
- [ ] 발생 시점과 발견 경위
- [ ] 요구되는 대응 내용

#### Step 2: 즉시 대응 조치 (10분 이내)
```
⏰ 소요시간: 5분

1. 관련자 연락:
   📞 삼우 담당자에게 전화
   "A-X23~X30 구간 Shop Drawing 변경 건으로 연락드립니다.
    현재 상황과 변경 사유를 확인하고 싶습니다."

2. 현장 상황 확인:
   📞 현장 관리자에게 연락
   "해당 구간 작업 현황과 영향도를 확인해주세요."

3. 초기 대응 기록:
   Google Sheet에서 해당 행 찾기
   - "처리담당" 컬럼에 본인 이름 입력
   - "처리상태"를 "검토중"으로 변경
   - "메모" 컬럼에 "삼우 담당자 연락 완료, 현장 확인 중" 입력
```

**✅ 즉시 대응 체크리스트**
- [ ] 발생원 담당자 연락 완료
- [ ] 현장 상황 확인 요청 완료
- [ ] 처리 담당자 지정 완료
- [ ] 초기 대응 내용 기록 완료

#### Step 3: 상황 분석 및 대응 계획 수립 (30분 이내)
```
⏰ 소요시간: 15분

1. 정보 수집 결과 정리:
   - 삼우 담당자 답변 내용
   - 현장 관리자 확인 결과
   - 변경 사항의 기술적 영향도

2. 대응 방안 검토:
   - 즉시 조치 가능한 사항
   - 추가 검토가 필요한 사항
   - 다른 업체와의 협의 필요 사항

3. 처리 계획 수립:
   - 단기 대응 (당일 내)
   - 중기 대응 (주 내)
   - 관련자별 역할 분담
```

#### Step 4: 상태 업데이트 및 공유
```
⏰ 소요시간: 5분

1. Google Sheet 업데이트:
   - "처리기한": 2025-01-17 (2일 후)
   - "처리상태": "조치중"
   - "메모": 상세한 대응 계획 기록
   - "비고": 관련자 연락처 및 참고사항

2. 관련자에게 상황 공유:
   📧 이메일 또는 메신저로 현황 공유
   "A-X23~X30 구간 이슈 대응 현황을 공유드립니다..."

3. Dashboard에서 이슈 상태 확인:
   - 해당 기둥들이 여전히 빨간색인지 확인
   - 필요시 수동 동기화 실행
```

**✅ 대응 완료 체크리스트**
- [ ] 상황 분석 완료
- [ ] 대응 계획 수립 완료
- [ ] 처리 상태 업데이트 완료
- [ ] 관련자 공유 완료

## 튜토리얼 3: 새로운 이슈 등록하기

### 📋 상황 설정
**현장에서 "B-X15 기둥에서 HMB와 PSRC 접합부 간섭 발견"이라는 연락을 받았습니다.**

### 🎯 학습 목표
- 수동으로 새 이슈를 등록하는 방법 익히기
- 적절한 긴급도와 분류 설정하기
- 관련 기둥 범위 지정하기

### 📱 단계별 실습

#### Step 1: 이슈 등록 준비
```
⏰ 소요시간: 3분

1. 현장 담당자로부터 상세 정보 수집:
   📞 "정확한 위치가 B-X15 맞나요?"
   📞 "주변 기둥들도 영향받나요?"
   📞 "언제 발견되었고, 작업에 미치는 영향은?"
   📞 "사진이나 도면이 있나요?"

2. 수집된 정보 정리:
   - 위치: B-X15 (단일 기둥)
   - 문제: HMB-PSRC 접합부 철근 간섭
   - 발견: 2025-01-15 14:30
   - 영향: 콘크리트 타설 지연 우려
```

#### Step 2: Google Sheet에 수동 등록
```
⏰ 소요시간: 5분

1. Google Sheet 열기
2. 가장 아래 빈 행으로 이동
3. 다음 정보 입력:

   A열(NO): 자동 번호 + 1
   B열(상태): 미처리
   C열(긴급도): High
   D열(발생원): 현장발견
   E열(공법구분): HMB-PSRC접합
   F열(메일ID): MANUAL_20250115_001
   G열(발신자): 현장관리자
   H열(수신일시): 2025-01-15 14:30
   I열(제목): [현장발견] B-X15 HMB-PSRC 접합부 간섭
   J열(본문요약): B-X15 기둥에서 HMB와 PSRC 접합부 철근 간섭으로 콘크리트 타설 불가
   K열(AI분석): 수동 등록 - 현장 확인 필요
   L열(추천조치): 접합부 상세 검토 및 철근 배치 조정
   M열(키워드): HMB, PSRC, 접합부, 간섭
   ...
   T열(처리담당): 본인 이름
   U열(처리기한): 2025-01-16
   V열(처리상태): 검토대기
   Z열(등록일시): 현재 시간
```

#### Step 3: Dashboard에서 확인
```
⏰ 소요시간: 2분

1. Dashboard로 이동
2. "🔄 지금 동기화" 버튼 클릭
3. B-X15 기둥 위치 확인:
   - 검색창에 "B-X15" 입력
   - 해당 기둥이 주황색(High)으로 표시되는지 확인

4. 우측 이슈 패널에서 새 이슈 확인:
   - "[현장발견] B-X15 HMB-PSRC 접합부 간섭" 항목 확인
```

#### Step 4: 후속 조치 계획
```
⏰ 소요시간: 3분

1. 즉시 조치 사항:
   - 해당 구간 작업 일시 중단 지시
   - 구조 설계팀에 검토 요청
   - 관련 도면 및 자료 수집

2. 관련자 연락:
   📞 구조 설계 담당자
   📞 현장 시공 담당자
   📞 품질 관리 담당자

3. 추적 관리:
   - 일일 진행 상황 확인
   - 해결 방안 도출 시까지 모니터링
   - 유사 사례 재발 방지 대책 수립
```

**✅ 이슈 등록 완료 체크리스트**
- [ ] 현장 정보 수집 완료
- [ ] Google Sheet 수동 등록 완료
- [ ] Dashboard 동기화 및 확인 완료
- [ ] 후속 조치 계획 수립 완료

## 튜토리얼 4: 주간 리포트 작성하기

### 📋 상황 설정
**매주 금요일, 한 주간의 이슈 현황을 정리하여 관리자에게 보고해야 합니다.**

### 🎯 학습 목표
- 주간 데이터 추출 및 분석 방법 익히기
- 효과적인 리포트 구성 방법 배우기
- 트렌드 분석 및 개선 제안 작성하기

### 📱 단계별 실습

#### Step 1: 주간 데이터 추출
```
⏰ 소요시간: 10분

1. Google Sheet에서 데이터 필터링:
   - 수신일시 컬럼(H열) 선택
   - 데이터 → 필터 만들기
   - 날짜 범위: 2025-01-13 ~ 2025-01-17 (월~금)

2. 기본 통계 수집:
   - 전체 신규 이슈 개수
   - 긴급도별 분포 (Critical, High, Medium, Low)
   - 발생원별 분포 (삼우, ENA, 이앤디몰, 센코어)
   - 공법구분별 분포 (PSRC-PC접합, HMB 등)

3. 해결 현황 파악:
   - 처리상태가 "조치완료"인 이슈 개수
   - 여전히 "검토중"인 이슈 개수
   - 지연되고 있는 이슈 (처리기한 초과)
```

#### Step 2: Dashboard에서 시각적 분석
```
⏰ 소요시간: 5분

1. Dashboard 전체 현황 스크린샷:
   - 전체 그리드 화면 캡처
   - Zone별 이슈 분포 확인
   - 색상별 기둥 개수 파악

2. 이슈 패널 현황 정리:
   - 활성 이슈 목록 스크린샷
   - 우선순위별 정렬 상태 확인
   - 장기 미해결 이슈 식별

3. 통계 정보 수집:
   - 하단 통계 패널 정보 기록
   - 전체 기둥 대비 이슈 비율 계산
```

#### Step 3: 리포트 작성
```
⏰ 소요시간: 15분

📊 주간 이슈 현황 리포트 (2025.01.13~01.17)

■ 요약
- 신규 이슈: 12건 (전주 대비 +3건)
- 해결 완료: 8건
- 진행 중: 15건
- 지연 이슈: 2건

■ 긴급도별 현황
- Critical: 2건 (PSRC 접합부 간섭, Shop DWG 변경)
- High: 5건 (HMB 반입 지연, 철근 간섭 등)
- Medium: 8건
- Low: 2건

■ 발생원별 분석
- 삼우(원설계): 4건 (설계 변경 관련)
- ENA(시공): 3건 (현장 시공 이슈)
- 이앤디몰(PC): 2건 (PC 부재 관련)
- 센코어(전환설계): 1건
- 현장발견: 2건

■ 주요 이슈 및 대응 현황
1. [Critical] PSRC-PC 접합부 간섭 (A-X23~X30)
   - 현황: 삼우 설계 변경으로 인한 철근 간섭
   - 대응: 접합부 상세 재검토 중
   - 예상 해결: 01.20

2. [High] HMB 반입 지연 (B-X10~X15)
   - 현황: 제작사 일정 지연
   - 대응: 대체 공급업체 검토 중
   - 예상 해결: 01.25

■ 트렌드 분석
- 접합부 관련 이슈 증가 추세 (전주 2건 → 이번주 5건)
- 삼우 발생 이슈 비중 높음 (전체의 33%)
```
