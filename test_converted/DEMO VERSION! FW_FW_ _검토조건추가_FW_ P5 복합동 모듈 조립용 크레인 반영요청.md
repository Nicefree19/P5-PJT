P5 복합동 모듈 조립용 크레인의 위치, Embedded Plate 위치 및 반력값을 송부드리오니 검토 및

추후 Shop DWG 반영요청드립니다.

 

1. 위치 : 5층 상단 (6층의 PC Girder) 설치 J~K열 27~29열 / 30~31열,  I~J열 33~34열 / 35~37열

2. 반력값 : 첨부파일 참조

- 80(20x4) Ton : 각 포인트당 530kN

- 10(5x2) Ton : 각 포인트당 74kN

3. 검토조건

Dead Load + Construction Load + 반력값

※ P6구간이므로 Live Load는 고려안하는 조건으로 검토바랍니다.

4. Embedded plate 상세위치 : 첨부파일 참조
