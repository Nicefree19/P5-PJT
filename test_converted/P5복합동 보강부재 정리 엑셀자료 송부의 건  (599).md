요청 자료 중 엑셀자료 먼저 송부드립니다.

해당 엑셀파일의 '전체 보강부재 summary'시트는 오늘 회의 중 제안되었던 양식에 맞게 정리하였으며,

기존 회의자료 수정 시 마지막에 같은 내용으로 첨부될 예정입니다.

엑셀 파일 내 'PC\_접합부 강도비교'시트는 요청하신 PC거더와 접합부의 강도비교 내용이 정리된 자료입니다.

summary 시트를 정리하던 중 다음의 거더들

3PG61, 3PG71, 3PG71e, 3PG72, 3PG72e, 3PG73, 3PG74, 3PG92, 4PG41, 4PG42, 4PG43, 5PG41, 6PG41, 7PG12, 10PG41, 10PG43

에서 쎈에서 정리한 자료와 수량 차이가 발생합니다.

(저희 측에서 이번에 작성한 자료는 거더의 개수를 바탕으로 수량을 입력하여 실제 접합부의 개소 숫자는 x2배를 하여야 합니다.)

해당 차이가 rev38 모델 이후 변경사항에 따른 차이인지 확인 부탁드립니다. 특히 4PG43 접합부 보강 내용이 쎈에서 정리한 자료에 없는데 해당 부재 이름 자체가 없어진 것인지 단순 누락인지 확인 요청드립니다.

첨부파일 참조 부탁드립니다.
