## 1. P5 단독 PRE-CON 검토 시 주요 고려사항

1. **P6 구간 적용 범위 확정**  
   * P5 단독 해석 시, P6 영역을 4F바닥 까지 적용하면 되는지 확인 요청.
   * 기존 P6구간에 적용되었던 9F 소방수조 무시하고 진행하면 되는지 확인 요청.
2. **기둥 축소 검토(고강도 재료 적용)**  
   * 고강도 콘크리트(60MPa)와 **고강도 강재(SM420)** 적용  
     
   * 기 제작된 PSRC기둥 정확한 영역파악 필요. **축소 미적용 구간**을 구별하고, 설계 모델에 반영해야함. (\* 기제작된 PC도 확인 필요. )  
     
   * 기존 제작 기둥 및 PC보에 대한 제작 현황을 미고려 ( 기존 제작분 폐기) 하여 진행할 것인지 확인 요청.
3. **P5-P6 경계부 기둥(C1B 등)의 이중 해석**  
   * “P5 단독” 시 외곽기둥 역할로 분담하중이 작아질 수 있으나, 상부층에서 모멘트 증가가 예상됨.
   * 향후 “P5+P6 통합” 시 다시 내부기둥이 되므로, **독립 모델 & 통합 모델** 양쪽에서 축력·모멘트 변화를 모두 검토 필요.
4. **지붕층 하중 정보 확정**  
   * 옥상 냉각탑, 수조, 스크러바, 파이프랙, 크레인 등  하중에 대한 정보가 방대하여, **최대치 또는 최종안에 대한 지붕층 적용 하중 정보 제공 요청.**
5. **설계 Iteration 최소화**  
   * 이앤디몰(협력사)과 **기둥 사이즈**, **보(PC/HMB/PSRC) 단면**등 주요 설계 파라미터를 사전에 조율한 뒤 해석을 진행해야 합니다.

   --> 즉, 사전에 PSRC기둥 축소가능 부위에 대해 사전에 계획하여 설계 반복횟수를 최소화필요.

---

## 2. 통합된 핵심 검토 포인트

1. **해석 모델 시나리오**  
   * **(A) P5 단독 모델(L자형)**: P6 구간을 부분적으로만 고려하는 모델 등.
   * **(B) P5+P6 통합 모델**: 경계부 기둥, 상부층 모멘트·응력 변화 고려 , 그외 기둥 반력 증가 유/무 확인(기초 영향성 고려)
2. **기둥∙보 단면 검토 및 응력비 평가**  
   * 고강도 재료(60MPa, SM420) 적용 시 **축소된 기둥 단면**이 해석 상 요구 내력·변위·시공성 조건을 만족하는지 확인.
   * PC보 / HMB 단면 재검토
3. **옥상 뜬바닥(수조)·냉각탑 하중 반영**  
   * 확정된 지붕층 하중 (증가 예상됨.)
   * **삼우설계 횡변위 기준(H/500)** 충족 여부 검토
4. **기초 검토 ( 필요시 : 검토 예상 시나리오. 기둥은 감소하였으나, 보 사이즈 증가에 따른 골조 자중 증가 )**  
   * 지붕층 하중 증가 , 기둥단면 축소--> 만약 보사이즈 증가 발생 시 기초 두께 및 전단 내력 부족 우려가 있으므로 추가적인 검증이 예상됨.
   * P5+P6 통합 모델 기반에서 1층 기둥 반력 기존대비 증가 여부 확인 필요.
5. **설계 일정 및 협업 구조**  
   * 3주 내 1차 해석 결과 보고(목표: 3/14).
   * 이앤디몰(설계협력사)과 지붕 **하중증가 , 기둥 축소 단면 등을 사전에 확정**해 설계 반복작업을 최소화.
