02/10 금요일 까지 ( 명일 )

1.  P5 + P6 복합동  파일 수량

  -  물산 대비 파일 증/감 파악 목적

   물산 검토 내용 - ( D600 PHC PILE , 설계 지지력 280Ton/본, 평균심도 37m )  --> P4 때 품질 테스트 결과 평균 316Ton/본 이 확인되어 설계 지지력 280 Ton /본 적용

  물산 예상 파일 수량 ( P5: 12017 공 + P6 : 9480 공 ) = 21497 공

--> 근영과장님 물산 예상 파일 수량과 대비하여 정리 부탁드립니다.  ( P5 수량 , P6 수량 , 총 수량  . 증가 수량 몇개 )

2. 기초 두께  (예상 두께 3m : 전단보강 적용 시 )  --> 저희공법 적용시 기초두께  확인 ( 전단보강 적용 가정하여 두께 산정 )

3. LFC + LFG  상세와 같은 HMB 철근 배근 상세도 ( 평면, 입면도 작성 )

--> 이건 저희가 작성하겠습니다.

4. 탕정때 기둥 외곽 슬래브가 나갈때 WIND BEAM 적용 사례 확인 요청.

--> 이인호 이사님 자료 잘 받았습니다. 이걸로 정리해서 보내겠습니다.
