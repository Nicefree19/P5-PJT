아래의 공사 시 검토 요청드립니다.

1. 5, 8, 10층 F열 거더 시공하중 검토

 - 검토 상황 : P6영역에서 모듈 조립 후 P5로 운송 설치

 - 검토 부재 : F열 거더 및 HBM, 기둥 (필요시)

                  ( 상세 위치는 PPT의 2~4 페이지 참조 )

 - 검토 하중 :  5층 Live 30 kPa

                   8층 Live 15 kPa

                   10층 Live 20 kPa 

 -  회신 :  8월 29일

2. 모듈 조립용 호이스트 Embedded Plate 검토

 - 자세한 사항은 기 송부드린 메일 참조 ( 김준영 프로 발송 )

 - 회신 :  8월 29일

3. 40 ton 크레인 운전시 검토 

 - 자세한 사항은 기 송부드린 메일 참조  ( 김준영 프로 발송 )
