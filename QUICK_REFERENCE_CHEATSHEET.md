## 🎯 색상 코드 및 우선순위

### Dashboard 그리드 색상
```
🔴 빨간색: Critical/Showstopper → 즉시 대응 (5분 이내)
🟠 주황색: High → 당일 대응 (4시간 이내)
🟡 노란색: Medium → 계획적 대응 (2일 이내)
🟢 초록색: 정상 상태 → 모니터링
⚫ 회색: 해당 없음/완료
```

### 긴급도 기준
```
Showstopper: Shop Drawing 완료 후 변경, 0.75fpu 오류
Critical: 변단면 설계 오류, 구조적 안전 문제
High: 접합부 간섭, 현장 작업 중단 필요
Medium: 설계 문의, 일반적 검토 요청
Low: 행정 연락, 정보 공유
```

## 🔧 자주 사용하는 기능

### 이슈 등록 (수동)
```
1. Google Sheet 마지막 행으로 이동
2. 필수 정보 입력:
   - 긴급도, 발생원, 제목, 처리담당
3. Dashboard에서 "🔄 지금 동기화"
4. 해당 기둥 색상 변경 확인
```

### 상태 업데이트
```
1. 해당 행 찾기 (메일ID 또는 제목 검색)
2. 처리상태 변경: 검토중 → 조치중 → 조치완료
3. 메모 컬럼에 진행 상황 기록
4. 완료 시 비고에 최종 해결 방안 기록
```

### 주간 리포트 데이터 추출
```

3. 발생원별 분포 확인
4. 처리상태별 현황 파악
```

## 🚨 문제 해결 빠른 가이드

### "새 메일이 분석되지 않음"
```
✅ 확인사항:
- 메일 제목에 키워드 포함 여부
- 발신자가 등록된 참여자인지
- 최근 분석 시간 확인 (Sheet 마지막 행)

🔧 해결방법:
- 관리자에게 수동 분석 요청
- 키워드/참여자 추가 요청
- 임시로 수동 등록
```

### "Dashboard 로딩 안됨"
```
✅ 확인사항:
- 인터넷 연결 상태
- 브라우저 버전 (Chrome 권장)
- 캐시 및 쿠키 상태

🔧 해결방법:
- 새로고침 (Ctrl+F5)
- 캐시 삭제
- 다른 브라우저 시도
- 관리자 문의
```

### "데이터 동기화 안됨"
```
✅ 확인사항:
- 네트워크 연결 상태
- 마지막 동기화 시간
- 다른 사용자도 같은 문제인지

🔧 해결방법:
- "🔄 지금 동기화" 버튼 클릭
- 페이지 새로고침
- 잠시 후 재시도
```
