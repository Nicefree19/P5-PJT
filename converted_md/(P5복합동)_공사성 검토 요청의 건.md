\\66.3.218.106\P5 Pre-AFD\P5 복합동 (협력사)\구조\12\_공사성검토\20230331\_검토요청및자료

 

1. Mega deck (모듈반입용)  

     설치위치 A/K,
32~33(5층, 11층확정)

      설치위치 A/K,
8~9(5층, 11층확정)

    - 기존 계산서에서 하중값 고려하시고 PSRC 기둥에 반영 가능 유무 확인 

    - PSRC 기둥에 접합부에 대한 철골 및 임베디드 등 선반영 방안 제출

    - 단부에 슬래브 돌출부와 간섭사항 고려 필요

    \* 서버에 업로드 자료: 기존 P3의 Mega deck 구조검토서, 접합상세(선반영해서 기둥에 철골부재 달아서 와서 서플라이스 접합함)

 

2.  대형리프트  

   - 기초 간섭(건물-리프트)사항 검토 (복합동 기초도면 없으므로 예상치로 확인)

   - 건물골조에 접합부 검토 (단부에 슬래브 돌출부와 간섭사항 고려 필요)

  \* 골리앗 리프트 기초 및 반력, 도면 자료 -  섭버에 업로드

3. 옥상 설치용 Luffing 크레인  (공사후 철거 기준)

   - 크레인 좌대의 반력값으로 건물 기둥에 추가시 구조적인 문제가 없는지 검토 

   - 크레인 설치 위치(사이드)로 인한 편심도 고려하여 상기항 검토

   - 앵커 설치 검토 (고정 방법 검토) 

   \* 크레인좌대 구조검토서, 양중계획도 -  섭버에 업로드

4. 인,화물용리프트 

   - PSRC 기둥에 반영 가능 유무 확인

   - 기초 간섭 될 것으로 예상됨 -> 방안 검토

   -  건물골조에 접합부 상세 검토 (단부에 슬래브 돌출부와 간섭사항 고려 필요)

   \* 별도 업로드 자료는 없음, ppt 내용 참조

5. 에스컬레이터 

  - 설치위치 간섭사항 검토 (기초, 부재접합부)

  - 기초는 자료 접수시 검토(추후), 

  - 건물 골조와 접합부 방안

  \* 별도 업로드 자료는 없음, ppt 내용 참조

6,7,8항은 검토 필요사항 없음 

\* P4 엠베디드 도면을 참조 자료로 업로드 했습니다. 참조하시기 바랍니다.
